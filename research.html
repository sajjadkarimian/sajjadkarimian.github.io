Research Projects

My work focuses on healthcare data transparency, trustworthy AI, ontology engineering, and socio-technical systems analysis.

Atlas Project

Data Transparency Ontology

Mapped healthcare data lifecycles using data flow diagrams, UML use cases, and data value mapping to support ontology-based transparency and provenance modelling.

SEIPS Work System Project

Healthcare Work System Ontology

Developed an ontology based on the SEIPS model to represent healthcare work systems, actors, tasks, tools, and socio-technical context around data use.

VICTORY Project

Moodle-Based Learning Platform

Built and customised a Moodle-based learning management system, including self-developed plugins to improve usability, structure, and course delivery.
